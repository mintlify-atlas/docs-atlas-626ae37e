## Overview

Eventos is a lightweight REST API that powers the UTMACH events platform. It provides a single, focused endpoint to retrieve all registered participants from a PostgreSQL database, with CORS protection and SSL-secured database connectivity.

## Key features

<CardGroup cols={2}>
  <Card title="Participant listing" icon="list">
    Retrieve all registered event participants with a single GET request.
  </Card>
  <Card title="PostgreSQL integration" icon="database">
    Persistent storage with a connection pool for efficient, scalable queries.
  </Card>
  <Card title="CORS protection" icon="shield">
    Requests are restricted to the official events frontend at `eventosutmach.000webhostapp.com`.
  </Card>
  <Card title="SSL-secured connections" icon="lock">

## Tech stack

| Layer | Technology |
|---|---|
| Runtime | Node.js 14.x |
| Framework | Express 4.x |
| Database | PostgreSQL (via `pg` pool) |
| Config | `dotenv` |
| CORS | `cors` package |

<Note>
  The API is designed to serve the UTMACH events frontend. The CORS policy restricts access to `https://eventosutmach.000webhostapp.com` by default.
</Note>
